# AI NEXT PHASE PROMPT

Focus: Improve the user experience of the Governed AI system while fixing integration weaknesses.

Primary UX goal:
The system must feel calm, trustworthy, and legible.

User must instantly understand:
- system health
- active signals
- pending approvals
- results of decisions
- resulting knowledge

Technical priorities:
1. Signal durability
2. Request / correlation IDs across systems
3. Health endpoints
4. Removal of hardcoded endpoints
5. Clear discovery → proposal → approval → knowledge flow

The system must visibly demonstrate the loop:

Discovery → Governed Capability → Receipt → Knowledge
